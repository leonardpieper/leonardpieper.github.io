Ceciplan ist eine Vertretungsplan App, die erst als kleines Projekt startete und mit der Zeit immer größer wurde. Neben dem Vertretungsplan bietet sie auch eine Kommunikationsplattform, auf der Lehrer mit Schülern kommunizieren- und Dateien austauschen können. Derzeit werden Android, Web und eine iOS WebApp angeboten. Ceciplan hat über 200 wöchentlich aktive Nutzer.

Benutzte Technologien

  • Android Studio
  • Java
  • HTML
  • JavaScript
  • NodeJS

Derzeit arbeite ich gemeinsam mit dem Land NRW an einer mobilen Fassung des Laufbahnberatungs- und Planungstool Oberstufe (kurz LuPO). Es handelt sich hierbei um eine Software, die den Schülerinnen und Schülern die Wahl ihrer Abiturfächer erleichtern soll.
Wichtig ist hierbei, dass nur Fächer und Fächerkombinationen gewählt werden dürfen, die vorher in der Prüfungsordnung festgelegt wurden.
